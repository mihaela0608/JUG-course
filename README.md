# 🚀 JUG Course - Java Practice Repository

This repository contains my personal practice, exercises, and mini-projects from the **Bulgarian Java User Group (JUG)** course.

The goal of this repo is to strengthen my understanding of core Java concepts through hands-on coding, problem solving, and real examples inspired by the course.

---

## 📚 About the Course

The Bulgarian Java User Group (JUG) course focuses on modern Java development practices, including:

- Core Java fundamentals
- Object-Oriented Programming (OOP)
- Collections & Generics
- Streams API
- File handling
- Clean code practices
- Real-world coding examples

This repository follows the learning path of the course and contains my own implementations and experiments based on the topics covered.
